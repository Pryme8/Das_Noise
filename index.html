Pryme8's Noise

This is a project to get all the fancy noises out there under one roof... I was getting super annoyed having to call different noises with differnt constructors, so I figured lets make this conform to my head. Most are based off of others public examples I'll make sure to credit them, but for the most part this is a whole new reworking.
Quite a few bugs for right now, and im pretty sure the way im doing the seed calculations are prtty hacky... Umm a few of the calculations are my own so my bad if they syck?
Once the project is done, there will be a ton of different ways to generate noise, and I will be porting a bunch of GLSL noises I did over as well along with my Worley System and others that I can find.
Look at the examples for how to work these buggers.
var noise = new Teriable.Noise(type => string ,seed => string, args => {});
noise.getValue(args =>{});

Mixing

Mix
noiseA = Noise('Simple2','SeedMe', {scale:20})
noiseB = Noise('Simple2','SeedMe2', {scale:20})
Mix(noiseA, noiseB, 0.75)
